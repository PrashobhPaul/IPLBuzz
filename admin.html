🌐 Cloudflare Worker Proxy

Why you need this: CricAPI blocks direct browser requests (CORS). A Cloudflare Worker acts as a middleman.

Setup:
1. Go to workers.cloudflare.com → Create Worker
2. Paste the code from worker.js in your repo
3. Deploy → copy the URL and paste it above
4. The worker forwards requests: /proxy?url=https://api.cricapi.com/...

🔐 Firebase Rules

Set your Firebase Realtime Database rules to:

{
  "rules": {
    ".read": true,
    ".write": "auth != null"
  }
}

This allows public reads (index.html works for all) and writes only when admin is logged in.
